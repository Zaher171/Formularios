# Proyecto BrandName - Sistema de Registro y Login

## Descripción
Este proyecto consiste en un sistema completo de **registro y login de usuarios** para la web BrandName, desarrollado con HTML, CSS y JavaScript. La autenticación y almacenamiento de datos se realiza mediante **Firebase Authentication** y **Firestore**.  

El sistema incluye:

- Formulario de **registro** con validación de correo, doble contraseña y datos adicionales (nombre, dirección, año de nacimiento).
- Formulario de **login** con validación de correo y contraseña.
- Botón para **mostrar/ocultar contraseña** (JavaScript).
- Registro de la **fecha y hora de acceso** de los usuarios.
- Cumplimiento opcional con la **LOPD** para la gestión de datos personales.
- Estética consistente y responsive basada en **Flexbox y Grid**, inspirada en Apple.com.

---

## Tecnologías utilizadas

- **HTML5 & CSS3**
- **JavaScript (ES6 Modules)**
- **Firebase**

  - Firestore
